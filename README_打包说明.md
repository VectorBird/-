# 打包说明

## 快速打包

1. **安装依赖**（如果还没安装）：
   ```bash
   pip install -r requirements.txt
   ```

2. **运行打包脚本**：
   - Windows: 双击运行 `build.bat`
   - 或者手动运行: `python build_exe.py`

3. **等待打包完成**（可能需要几分钟）

4. **找到生成的EXE文件**：
   - 位置: `dist/抖音直播中控控场工具V1.1版本.exe`

## 打包后的文件说明

- **EXE文件大小**: 通常在100-200MB左右（包含Qt WebEngine浏览器内核）
- **首次启动**: 可能需要几秒钟时间
- **配置文件**: 程序会在运行目录自动创建 `danmu_cfg.json` 配置文件
- **日志文件**: 日志文件会保存在程序运行目录的 `logs/` 文件夹中

## 注意事项

1. **杀毒软件**: 部分杀毒软件可能会误报，需要添加信任
2. **首次运行**: 首次运行可能需要一些时间来初始化
3. **依赖**: EXE文件是独立的，不需要安装Python即可运行
4. **数据目录**: 程序会在EXE所在目录创建配置文件和数据文件

## 如果遇到问题

1. 检查是否有足够的磁盘空间（至少500MB）
2. 确保所有依赖都已正确安装
3. 查看打包过程中的错误信息
4. 检查是否有杀毒软件阻止
